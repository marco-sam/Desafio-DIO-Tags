História da Web

(Voltar)

Os primeiros computadores foram gerados na década de 1940. Em geral, estes eram de uso militar e encarregados de realizar complexos cálculos.

Com o passar das gerações de computadores, seus tamanhos foram diminuindo drasticamente. Um computador de 1ª geração ocupava uma sala inteira, enquanto que um computador de 3ª geração (Déc. 1970) não ocupava mais que 1m².

"Vivemos atualmente a 5ª geração dos computadores." Esta geração é marcada pela virtualização dos processos humanos
. a citar por exemplo, o Metaverso e o uso de criptomoedas.

Alguns nomes importantes neste contexto são: Tim Benners Lee, como o criador do WWW, Bill Gates como fundador da Microsoft e por gerar acessibilidade ao acesso a internet, e Steve Jobs, famoso como fundador da Apple e do Iphone, é conhecido com o visionário que trouxe ao mundo a utilização distribuída dos smartphones.

Comunicação Client x Server

(Voltar)

No contexto Web, um Client pode ser definido como um ator que consome uma informação proveniente de um lugar(server).

Uma máquina server é uma máquina cuja principal função é hospedar um tipo de serviço (informação, arquivo, firewall, streaming, e-mail etc.).

O ciclo de vida deste serviço é: O server hospeda o serviço: O serviço é enviado aos clients através do protocolo HTTP (TCP/IP); Os clients recebem o serviço em dispositivos como PCs e Smartphones.

Dispositivos como Smartphones, Notebooks, Tablets e Smartwatches podem ser utilizados como clients. Maquinas server em geral são constituídos apenas por um gabinete com CPU.

Introdução HTML

(Voltar)

Antes de tudo é importante ressaltar que HTML não é linguagem de programação. Como seu próprio nome informa, HTML é uma linguagem de marcação.

É utilizado para realizar as marcações de linguagem para web, podendo ser interpretada e acessada através de navegadores web como Mozila firefox e Google chrome.

Sua estrutura básica é formada por tags sinalizadas por '<>', A estrutura utilizada é: < html >< head >< body > < / body>< / head>< / html>"

O HTML pode ser associado a uma página de estilos de extensão CSS, e/ou a uma página de interação de extensão JS, entre outros.
